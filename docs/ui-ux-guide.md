# AI Co-Learner UI/UX 디자인 가이드

## 1. 프로젝트 개요

### 서비스 설명
AI Co-Learner는 **AI 기반 학습 코칭 플랫폼**으로, 사용자가 AI 봇과 대화하면서 학습하고, 자동으로 6가지 핵심 역량을 분석받는 서비스입니다.

### 주 타겟층 및 우선순위
**1차 타겟: 청소년 (중·고등학생, 13-18세) - 60%**
- 학교 교육 보완 및 자기주도 학습 습관 형성
- 진로 탐색 및 미래 역량 개발
- 게이미피케이션을 통한 학습 동기 부여

**2차 타겟: 대학생 및 취준생 (19-25세) - 30%**
- 전공 학습 및 취업 준비
- 실무 역량 및 소프트스킬 개발
- 자기계발 및 경력 관리

**3차 타겟: 직장인 (26세 이상) - 10%**
- 업무 역량 향상 및 리스킬링
- 업계 트렌드 학습 및 전문성 강화

### 주요 과업 (Core Tasks)

#### 과업 1: AI 봇과 대화하며 학습하기 (가장 빈번)
- **빈도**: 일 1-3회
- **소요 시간**: 회당 10-30분
- **주요 행동**:
  1. 대시보드에서 봇 선택
  2. 학습 주제 관련 질문 입력
  3. AI 답변 읽고 추가 질문
  4. 학습 내용 정리/메모

#### 과업 2: 학습 진행 상황 확인하기
- **빈도**: 주 2-3회
- **소요 시간**: 3-5분
- **주요 행동**:
  1. 대시보드 역량 차트 확인
  2. 일일 퀘스트 진행 상황 체크
  3. 획득한 뱃지/레벨 확인
  4. 학습 통계 리뷰

#### 과업 3: 새로운 학습 목표 설정하기
- **빈도**: 주 1회
- **소요 시간**: 5-10분
- **주요 행동**:
  1. 부족한 역량 파악
  2. 추천 봇 확인
  3. 새 봇 생성 또는 기존 봇 선택
  4. 학습 목표 설정

#### 과업 4: 초기 역량 진단받기 (신규 사용자 전용)
- **빈도**: 최초 1회
- **소요 시간**: 15-20분
- **주요 행동**:
  1. 진단 문항 응답
  2. 결과 확인 (레이더 차트)
  3. 추천 봇 확인
  4. 첫 대화 시작

### 핵심 가치
- **개인화된 학습 경험**: 사용자의 역량에 맞춘 AI 봇 추천
- **자동 역량 분석**: 대화 내용을 분석하여 6가지 역량 자동 평가
- **게이미피케이션**: 퀘스트, 뱃지, 레벨 시스템으로 학습 동기 부여
- **자기주도 학습 지원**: AI와의 대화를 통한 능동적 학습 촉진

## 2. 타겟 사용자

### 주요 사용자 페르소나

#### 페르소나 1: 청소년 정호기 (16세, 고등학생)
- **목표**: 학교 공부, 진로 탐색, 자기계발
- **행동 패턴**:
  - 모바일 네이티브, 짧은 집중 시간
  - 게임적 요소에 민감, 즉각적인 피드백 선호
  - 친구들과 경쟁/공유 욕구
- **니즈**:
  - 재미있고 흥미로운 학습 경험
  - 눈에 보이는 성장 (레벨, 뱃지)
  - 간단명료한 인터페이스
- **특별 고려사항**:
  - 청소년 보호 정책 준수
  - 과도한 사용 방지 기능 (일일 사용 시간 제한 옵션)
  - 부모/교사 모니터링 기능 (향후)

#### 페르소나 2: 대학생 김학습 (22세)
- **목표**: 전공 지식 습득 및 역량 개발
- **행동 패턴**: 모바일로 짧은 시간 학습, 시각적 피드백 선호
- **니즈**: 빠른 응답, 직관적인 UI, 학습 진행 상황 확인

#### 페르소나 3: 직장인 박성장 (28세)
- **목표**: 업무 역량 향상 및 커리어 개발
- **행동 패턴**: 출퇴근 시간 활용, 실용적인 학습 선호
- **니즈**: 효율적인 학습 추적, 역량별 맞춤 추천

#### 페르소나 4: 취준생 이준비 (25세)
- **목표**: 취업 준비 및 면접 대비
- **행동 패턴**: 체계적인 학습 계획, 성취감 중요
- **니즈**: 명확한 학습 목표, 달성 가능한 퀘스트

## 3. 주요 화면 및 기능

### 3.1 로그인 페이지 (`/login`)
- **목적**: 사용자 인증 (AWS Cognito)
- **요구사항**:
  - 간결하고 신뢰감 있는 디자인
  - 이메일/비밀번호 입력
  - 회원가입, 비밀번호 찾기 링크
  - 로딩 상태 표시

### 3.2 초기 진단 페이지 (`/initial-assessment`)
- **목적**: 신규 사용자의 6가지 역량 초기 평가
- **요구사항**:
  - 프로그레스 바로 진행 상황 표시
  - 역량별 질문 (10-15문항)
  - 직관적인 답변 입력 (객관식/서술형)
  - 결과 시각화 (레이더 차트)

### 3.3 대시보드 (`/dashboard`)
- **목적**: 학습 현황 및 역량 한눈에 파악
- **핵심 요소**:
  1. **역량 차트** (Radar Chart)
     - 6가지 역량 시각화 (0-100점)
     - 전체 평균과 비교
     - 색상 코드로 강약점 표시

  2. **AI 봇 목록** (Bot Cards)
     - 봇 이름, 설명, 아이콘
     - 추천 봇 강조 (역량 기반)
     - 최근 대화 시간 표시
     - 새 봇 생성 버튼

  3. **일일 퀘스트** (Daily Quest)
     - 오늘의 학습 목표 (3-5개)
     - 완료 체크박스
     - 진행률 바

  4. **학습 통계**
     - 총 대화 횟수
     - 연속 학습 일수
     - 획득한 뱃지
     - 현재 레벨

### 3.4 채팅방 (`/chat/:botId`)
- **목적**: AI 봇과 실시간 대화
- **핵심 요소**:
  1. **헤더**
     - 봇 이름 및 아이콘
     - 뒤로가기 버튼
     - 설정 메뉴 (대화 초기화, 봇 삭제)

  2. **메시지 영역**
     - 사용자/봇 메시지 구분
     - 타임스탬프
     - 마크다운 지원
     - 코드 블록 하이라이팅
     - 로딩 애니메이션 (타이핑 효과)

  3. **입력 영역**
     - 멀티라인 텍스트 입력
     - 전송 버튼
     - 파일 첨부 (향후 기능)

### 3.5 학습 분석 페이지 (향후 기능)
- **목적**: 상세한 학습 패턴 분석
- **요구사항**:
  - 시간별/주제별 학습 통계
  - 역량 변화 그래프 (시계열)
  - AI 추천 학습 경로

### 3.6 청소년 보호 기능 (선택)
- **목적**: 건강한 학습 습관 형성 및 과몰입 방지
- **요구사항**:
  1. **일일 학습 시간 대시보드**
     - 오늘 사용 시간 표시
     - 권장 시간 대비 게이지
     - 휴식 시간 알림

  2. **학습 시간 제한 설정** (선택 사항)
     - 하루 최대 학습 시간 설정
     - 시간 초과 시 부드러운 종료 유도
     - "내일 또 만나요!" 메시지

  3. **부모/교사 대시보드** (향후)
     - 학습 진행 상황 모니터링
     - 역량 발달 리포트
     - 주간/월간 요약

## 4. 6가지 핵심 역량

### 역량 정의 및 아이콘 제안

| 역량 | 영문명 | 설명 | 아이콘 제안 | 색상 제안 |
|------|--------|------|------------|-----------|
| **질문력** | Question Quality | 명확하고 구조화된 질문을 하는 능력 | 💡 물음표, 전구 | #3B82F6 (파란색) |
| **사고력** | Thinking Depth | 깊이 있고 논리적인 사고 능력 | 🧠 뇌, 체스 | #8B5CF6 (보라색) |
| **창의력** | Creativity | 새롭고 독창적인 아이디어 생성 | 🎨 팔레트, 별 | #F59E0B (주황색) |
| **소통력** | Communication Clarity | 명확하고 효과적인 의사소통 | 💬 말풍선, 확성기 | #10B981 (초록색) |
| **실행력** | Execution Oriented | 계획을 실제로 실행하는 능력 | ⚡ 번개, 체크마크 | #EF4444 (빨간색) |
| **협업력** | Collaboration Signal | 타인과 협력하는 능력 | 🤝 악수, 연결 | #EC4899 (분홍색) |

## 6. 사용자 플로우

### 신규 사용자 플로우
```
1. 로그인/회원가입
   ↓
2. 초기 역량 진단 (15분)
   ↓
3. 진단 결과 확인 (레이더 차트)
   ↓
4. 대시보드 진입 (추천 봇 확인)
   ↓
5. 첫 번째 봇과 대화 시작
```

### 일반 사용자 플로우
```
1. 로그인
   ↓
2. 대시보드 확인
   - 오늘의 퀘스트
   - 역량 변화
   - 새 알림
   ↓
3. 봇 선택 또는 새 봇 생성
   ↓
4. 대화 진행
   ↓
5. 퀘스트 완료 및 뱃지 획득
```

## 8. 애니메이션 및 인터랙션

### 주요 애니메이션
1. **페이지 전환**: Fade + Slide (300ms)
2. **카드 호버**: 살짝 떠오름 + 그림자 증가
3. **버튼 클릭**: Scale down (100ms)
4. **차트 로딩**: 부드러운 그리기 애니메이션 (1초)
5. **메시지 입력**: 타이핑 효과 (점 3개 깜빡임)

### 마이크로 인터랙션
- 퀘스트 완료 시 체크 애니메이션 + 사운드 효과 (선택)
- 레벨업 시 폭죽 효과 + 축하 메시지
- 뱃지 획득 시 반짝임 효과 + 모달 팝업
- 연속 학습일 달성 시 불꽃 이모지 효과 (청소년 선호)

## 13. 디자인 미팅 질문 리스트

### UI/UX 디자이너에게 확인할 사항
1. **브랜드 아이덴티티**: 서비스 톤앤매너, 캐릭터 필요 여부
2. **타겟 우선순위**: 모바일 퍼스트 vs 데스크톱 퍼스트
3. **차별화 포인트**: 경쟁사 대비 특별한 UX 요소
4. **일러스트레이션**: 스타일 (미니멀, 일러스트, 3D 등)
5. **애니메이션 수준**: 미니멀 vs 풍부한 애니메이션
6. **다크 모드**: 1차 출시 포함 여부

### 청소년 사용자 관련 추가 질문
7. **청소년 친화적 디자인**:
   - 캐릭터/마스코트 도입 여부
   - 이모지, 스티커 활용 정도
   - 밝고 생동감 있는 컬러 vs 차분한 컬러
8. **게이미피케이션 수준**:
   - 뱃지/레벨 디자인 스타일 (귀여움, 쿨함, 현실적)
   - 리더보드/친구 비교 기능 포함 여부
9. **청소년 보호 기능**:
   - 학습 시간 알림 UI 디자인 (강제성 vs 권장)
   - 부모 모니터링 기능 UI (1차 출시 여부)
10. **연령대별 UI 차별화**:
    - 청소년/성인 사용자 UI 분리 여부
    - 프로필 설정에서 연령대 선택 후 맞춤 UI 제공

## 14. 청소년 사용자를 위한 특별 고려사항

### 디자인 원칙
1. **직관성 최우선**
   - 복잡한 메뉴 구조 지양
   - 한눈에 이해 가능한 아이콘과 레이블
   - 튜토리얼/온보딩 강화

2. **즉각적인 피드백**
   - 모든 액션에 대한 시각적 반응
   - 로딩 시간 최소화
   - 진행 상황 명확하게 표시

3. **긍정적 강화**
   - 실패보다는 성장 강조
   - 격려 메시지 및 응원 문구
   - 작은 성취도 축하

4. **안전한 환경**
   - 부적절한 콘텐츠 필터링
   - 개인정보 보호 강조
   - 신고 기능 명확히 배치

### 콘텐츠 톤앤매너
- **말투**: 반말 vs 존댓말 (연령대 고려)
- **메시지**: 짧고 명확한 문장
- **이모지 활용**: 적극 활용하되 과하지 않게
- **격려 문구 예시**:
  - "잘하고 있어요! 💪"
  - "오늘도 한 걸음 성장했어요! 🌱"
  - "3일 연속 학습 달성! 대단해요! 🔥"
