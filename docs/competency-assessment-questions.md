# 역량 평가 SJT 질문 20문항

AI Co-Learner 초기 진단 시스템에서 사용하는 **상황판단검사(SJT)** 기반 역량 평가 질문입니다.
사용자의 6가지 핵심 역량 **성향 프로파일**을 측정합니다.

---

## SJT (Situational Judgment Test) 방식이란?

기존 자기평가형("나는 어떤 사람인가?")과 달리, **구체적인 상황을 제시하고 행동 선택을 묻는 방식**입니다.

### 기존 방식과의 차이

| 구분 | 기존 (자기평가형) | SJT (상황판단형) |
|------|-----------------|----------------|
| 질문 형태 | "나는 어떤 사람?" | "이 상황에서 뭘 하겠는가?" |
| 정답 존재 | 있음 (4번이 항상 최고) | **없음** (모두 합리적인 선택) |
| 조작 가능성 | 높음 (패턴 파악 쉬움) | **낮음** (좋아 보이는 답이 없음) |
| 측정 방식 | 점수 합산 | **성향 프로파일링** |

### 핵심 설계 원칙

1. **정답 없음**: 모든 선택지가 합리적이며, 어떤 것이 "더 좋은 답"인지 알 수 없음
2. **성향 측정**: 선택 패턴을 통해 사용자가 어떤 역량을 자연스럽게 우선시하는지 파악
3. **이중 랜덤화**: 질문 순서 + 선택지 순서 모두 매번 셔플
4. **채점 정보 비공개**: 프론트엔드에는 선택지 텍스트만 전달 (역량 매핑 정보 숨김)

---

## 역량 매핑

각 선택지는 하나의 **주 역량(primary)** 과 **부 역량(secondary)** 에 매핑됩니다:

- **Q**: 질문력 (Question Quality)
- **T**: 사고력 (Thinking Depth)
- **C**: 창의력 (Creativity)
- **Co**: 소통력 (Communication Clarity)
- **E**: 실행력 (Execution Oriented)
- **Cl**: 협업력 (Collaboration Signal)

---

## 채점 방식

### 점수 부여
- **주 역량 (primary)**: 선택 시 **+3점**
- **부 역량 (secondary)**: 선택 시 **+1점**

### 최종 점수 계산 (기대값 기준 정규화)
```
총 배분 점수 = 20문항 × (3 + 1) = 80점
역량당 기대값 = 80 / 6 ≈ 13.3점 (균형 선택 시)

각 역량별:
  획득 점수 = 해당 역량의 primary(+3) + secondary(+1) 합계
  ratio = 획득 점수 / 기대값
  정규화 점수 = 1 + ratio × 4.5 → clamp(1, 10)
```

### 스케일 예시
| ratio | 의미 | 점수 |
|-------|------|------|
| 0 | 해당 역량 전혀 선택 안 함 | 1점 |
| ~0.5 | 다른 역량을 더 선호 | 3~4점 |
| ~1.0 | 균형 잡힌 선택 | 5~6점 |
| ~1.5 | 해당 역량 약간 선호 | 7~8점 |
| ~2.0+ | 해당 역량 강하게 선호 | 9~10점 |

### 결과 해석
- **9~10점**: 해당 역량을 강하게 우선시하는 성향
- **5~7점**: 균형적으로 활용하는 성향
- **3~4점**: 다른 역량을 더 우선시하는 성향
- **1~2점**: 해당 역량을 거의 선택하지 않은 경우

> **참고**: SJT는 "능력의 절대적 수준"이 아니라 "어떤 역량을 자연스럽게 선호하는지"를 측정합니다.
> 낮은 점수 = 능력 부족이 아니라, 다른 접근 방식을 더 선호한다는 의미입니다.

## 역량별 분포

### 주 역량(primary) 출현 횟수 (총 80 = 20문항 × 4선택지)

| 역량 | 출현 횟수 | 출현 문항 |
|------|----------|----------|
| 질문력 (Q) | 13회 | Q1-Q10, Q16-Q18 |
| 사고력 (T) | 13회 | Q1-Q6, Q11-Q14, Q16, Q18-Q19 |
| 창의력 (C) | 13회 | Q1-Q3, Q7-Q9, Q11-Q13, Q15, Q17, Q19-Q20 |
| 소통력 (Co) | 14회 | Q1, Q4-Q5, Q7-Q8, Q10-Q12, Q14-Q17, Q19-Q20 |
| 실행력 (E) | 14회 | Q2, Q4, Q6-Q7, Q9-Q11, Q13-Q16, Q18-Q20 |
| 협업력 (Cl) | 13회 | Q3, Q5-Q6, Q8-Q10, Q12-Q15, Q17-Q18, Q20 |

→ 6개 역량이 **13~14회로 균등 분포** (편차 ±1 이내)

| 선택지 | 주 역량 | 부 역량 |
|--------|---------|---------|

| 독창적이고 흥미로운 발표 구성을 먼저 구상한다 | C | Co |
